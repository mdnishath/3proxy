# keys/

Runtime secrets — NOT tracked in git. After cloning the repo, populate this
folder before running `fresh-setup.ps1`:

### `panel_id_ed25519`
OpenSSH ed25519 private key (matches `~/.ssh/authorized_keys` on the VPS as the
SSH user the tunnel script uses — usually `root`). Transfer via AnyDesk, scp,
or USB — never over clear channels, never into git. Ensure the file
permissions let only the Windows user + SYSTEM read it; `fresh-setup.ps1` and
`start-all.ps1` lock it down via `icacls`.

If you already ran a prior install on this PC, the key likely already lives at
`C:\proxy-farm\keys\panel_id_ed25519`; `fresh-setup.ps1` won't overwrite it.

### `agent-secret.txt`
Auto-generated by `fresh-setup.ps1` on first run — a 32-byte hex shared secret
the rotation agent requires in the `X-Agent-Secret` HTTP header. After
fresh-setup prints it, paste the value into the VPS panel's `.env` as
`ROTATION_AGENT_SECRET=...`.
